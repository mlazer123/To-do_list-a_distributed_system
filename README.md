# Todo App (FastAPI + Docker)

Небольшой учебный проект с разделением на frontend, backend и базу данных, упакованный в Docker.

## Описание

Это простое веб-приложение для работы со списком задач (todo):

* добавление задач
* получение списка задач

## Архитектура

Проект состоит из нескольких сервисов:

* **frontend** — статический сайт (nginx)
* **backend** — API на FastAPI
* **db** — PostgreSQL база данных

### Взаимодействие

1. Пользователь открывает сайт (`localhost:3000`)
2. Браузер загружает frontend
3. Frontend отправляет запросы к backend
4. Backend обрабатывает запрос и работает с БД
5. Результат возвращается обратно пользователю


## Docker

Все сервисы запускаются через **docker-compose**.

Каждая компонента работает в отдельном контейнере

## Запуск

### 1. Клонировать репозиторий

```
git clone <repo_url>
cd <repo>
```

### 2. Запустить контейнеры

```
docker compose up --build
```

### 3. Открыть в браузере

* Frontend: [http://localhost:3000](http://localhost:3000)
* Backend (docs): [http://localhost:8000/docs](http://localhost:8000/docs)
